1. Two Sum 
def twoSum(nums, target):
    prevMap = {} # val : index
    
    for i, n in enumerate(nums):
        diff = target - n
        if diff in prevMap:
            return [prevMap[diff], i]
        prevMap[n] = i

O(n) time O(n) space

lightbulb Logic Breakdown

Instead of checking every pair of numbers (which would be O(nÂ²)), we can check if the complement exists in a single pass.

We iterate through the array once. For each element n, we calculate diff = target - n. If diff is already in our hash map, we found the pair! If not, we store n and its index for future lookups.

Note: This approach trades space for speed by using a hash map to store visited elements.
